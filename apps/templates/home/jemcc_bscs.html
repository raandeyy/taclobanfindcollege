Program Overview

Bachelor of Science in Computer Science is an undergraduate program that is designed to help students be proficient in designing, writing, and developing computer programs and networks. Students will learn about computer algorithms and their implementation through computer software and hardware. The intricacies of software applications, data processing, web development, programming, and computer architecture will also be tackled in this program. BS in Computer Science is for students looking to pursue careers as a Software Developer, Web Developer, Database Administrator, Network Administrator, etc.
